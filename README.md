# Bryce Johnson ePortfolio

## CS 499 Computer Science Capstone

This repository contains my professional ePortfolio developed as part of the CS 499 Computer Science Capstone at Southern New Hampshire University.

The purpose of this ePortfolio is to demonstrate growth across key areas of computer science through the enhancement of previously developed artifacts. Each artifact has been revised to reflect improved design, functionality, and alignment with industry practices.

## Portfolio Structure

This ePortfolio is organized into the following sections:

- Informal Code Review  
- Software Design and Engineering Enhancement  
- Algorithms and Data Structures Enhancement  
- Databases Enhancement  

Each section includes original and enhanced artifacts along with a narrative explaining the improvements and lessons learned.
